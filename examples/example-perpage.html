DataTables HideEmptyColumns Plugin - Per Page

Description: Instead of hiding the column if every row is empty in a specific column (default), only hide the column if the column is empty for the current specific page. This means if a specific column is only populated on page 4, then you will only see that column on page 4, and no other pages.
