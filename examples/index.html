DataTables HideEmptyColumns Plugin

Author: Justin Hyland
Links: GIT Repository | DataTables | Author | Other Projects by Author | Other Cool Stuff

Description: Hide all (or specified) columns if no cells in the column(s) are populated with any values

HideEmptyColumns Examples

Note: All of these examples also use the columnsToggle button, which allows you to toggle the visibility of each cholumn. Please note that this is not required for hideEmptyColumns to work, I just included it so you can easily see which columns were disabled by the configuration being used.
  • Target all columns & Use colVis (colVis is optional)
  • Target columns by Indexes
  • Target Column Name (JSON or AJAX)
  • Change the column list type from Whitelist (default), to Blacklist (Targeting all columns except specified)
