DataTables HideEmptyColumns Plugin - Custom Defined Empty Values

Description: In addition to checking if the values are empty (empty, null, etc), you can add one or more custom values to be interpreted as empty, this is useful if you want to do something like only show the column if the values are anything but empty or other "falsey" values. You can specify a single string value, numeric value, or even a regular expression pattern, as well as specify an array of such patterns.
